# M5S07 Abstract & Conclusion

> Stage: M5S07 | Agent: Writing | Module: M5 Writing

---

## 1. Abstract（150-250 words）

### 结构检查

- [ ] **一句话概括**: 用一句话概括全文核心贡献
- [ ] **问题**: 本文解决什么具体问题？
- [ ] **方法**: 核心方法/洞察是什么？（避免技术细节堆砌）
- [ ] **主要结果**: 给出具体数值（如 "在 X 数据集上提升 Y%"）
- [ ] **贡献**: 本文的核心学术贡献

### 数值一致性验证

| 数值 | Abstract 中的表述 | 正文中的位置 | 是否一致 |
|------|-----------------|------------|---------|
| 主指标提升 | ... | M5S05 Table 1 | 是/否 |
| 数据集名称 | ... | M5S05 | 是/否 |
| baseline 名称 | ... | M5S05 | 是/否 |
| 其他关键数字 | ... | ... | 是/否 |

---

## 2. Conclusion（0.3 页）

### 结构检查

- [ ] 总结核心贡献（与 Abstract 一致，但不重复句子）
- [ ] 不引入新方法、新实验、新结果
- [ ] 可简要提及未来方向（1-2 句）
- [ ] 结尾有力，强化核心信息

---

## 写作检查清单

- [ ] Abstract 长度在 venue 要求范围内
- [ ] Abstract 可独立理解（不依赖正文）
- [ ] Conclusion 无新内容
- [ ] 全文所有数值交叉验证通过
- [ ] 术语全文一致
- [ ] 已遵循 M5S02 Style & Layout Profile 的摘要/结论语气与长度约束
- [ ] 无口语化表达
